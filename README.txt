To run MULSEA:
1. Adjust parameters in $\MULSEA-EXE\MultiMMHG.exe.config - see documentation in the file
2. Run MultiMMHG.exe

* Input samples can be found in $\MULSEA-EXE\Input-samples. 
  To run MULSEA with those samples, copy the "MULSEA" folder to C:\ and run MultiMMHG.exe (without changing the .config file above).
  The output will appear in folder C:\MULSEA\Output-samples.
